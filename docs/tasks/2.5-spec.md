## Objective

Create Objective-C CGO bindings for macOS Accessibility API to capture content from Terminal.app windows. This is the foundation for the macOS terminal adapter that will allow PairAdmin to automatically capture terminal content without requiring tmux.

---

## Requirements

### 1. Header File (`accessibility.h`)

Create Objective-C header file with C-compatible function declarations:

```c
// Initialize accessibility and request permissions
int initialize_accessibility();
int request_accessibility_permission();
int is_accessibility_enabled();

// Get Terminal.app window information
char** get_terminal_window_titles(int* count);
int* get_terminal_window_pids(int* count);

// Extract text from Terminal.app window
char* extract_text_from_window(int pid);
char* extract_text_from_frontmost_terminal();

// Memory management
void free_string_array(char** array, int count);
void free_string(char* str);
```

### 2. Implementation File (`accessibility.m`)

Implement functions using macOS Accessibility API (AXUIElement, NSAccessibility):

**Key APIs to use:**
- `AXUIElementCreateApplication(pid_t)` - Create accessibility element for process
- `AXUIElementCopyAttributeValue()` - Get attributes like "AXTitle", "AXFocusedWindow"
- `AXUIElementCopyAttributeValues()` - Get text content from textarea
- `NSAccessibilityProtocol` - Modern accessibility protocol

**Functions to implement:**

1. **`initialize_accessibility()`**
   - Check if accessibility is enabled
   - Return 0 on success, -1 on failure

2. **`request_accessibility_permission()`**
   - Trigger macOS permission dialog if needed
   - Open System Preferences → Privacy → Accessibility
   - Return 1 if granted, 0 if denied, -1 on error

3. **`is_accessibility_enabled()`**
   - Check `AXIsProcessTrusted()`
   - Return 1 if enabled, 0 if not

4. **`get_terminal_window_titles()`**
   - Find all Terminal.app processes
   - Get window titles via "AXWindows" → "AXTitle"
   - Return C string array

5. **`get_terminal_window_pids()`**
   - Get process IDs for Terminal.app windows
   - Return int array

6. **`extract_text_from_window(pid)`**
   - Create AXUIElement for process
   - Get focused window or first window
   - Extract text from textarea element
   - Return C string (caller must free)

7. **`extract_text_from_frontmost_terminal()`**
   - Get frontmost Terminal.app window
   - Extract text content
   - Return C string

8. **Memory management functions**
   - `free_string_array()` - Free array of strings
   - `free_string()` - Free single string

### 3. Go CGO Wrapper (`accessibility_cgo.go`)

Create Go wrapper that calls C functions:

```go
package macos

/*
#cgo CFLAGS: -x objective-c
#cgo LDFLAGS: -framework Cocoa -framework ApplicationServices
#include "accessibility.h"
*/
import "C"

// InitializeAccessibility initializes accessibility and requests permission
func InitializeAccessibility() error

// IsAccessibilityEnabled checks if accessibility is enabled
func IsAccessibilityEnabled() bool

// RequestAccessibilityPermission requests accessibility permission
func RequestAccessibilityPermission() (bool, error)

// GetTerminalWindowTitles returns titles of all Terminal.app windows
func GetTerminalWindowTitles() ([]string, error)

// GetTerminalWindowPids returns PIDs of Terminal.app windows
func GetTerminalWindowPids() ([]int, error)

// ExtractTextFromWindow extracts text from a Terminal.app window by PID
func ExtractTextFromWindow(pid int) (string, error)

// ExtractTextFromFrontmostTerminal extracts text from frontmost Terminal.app window
func ExtractTextFromFrontmostTerminal() (string, error)
```

## Implementation Details

### Accessibility Permission Flow

1. Check `AXIsProcessTrusted()` - if true, already granted
2. If false, call `AXIsProcessTrustedWithOptions()` with auto-prompt
3. If still false, open System Preferences with URL:
   `x-apple.systempreferences:com.apple.preference.security?Privacy_Accessibility`
4. User must manually enable in System Preferences

### Text Extraction Strategy

Terminal.app exposes content via:
1. Get application element: `AXUIElementCreateApplication(pid)`
2. Get windows: `AXUIElementCopyAttributeValue(app, "AXWindows", ...)`
3. Get focused window or first window
4. Get textarea element: `AXUIElementCopyAttributeValue(window, "AXFocusedTextArea", ...)`
5. Extract text: `AXUIElementCopyAttributeValue(textarea, "AXValue", ...)`

### Memory Management

- All C strings must be duplicated with `strdup()` before returning to Go
- Go must call `free_string()` or `free_string_array()` after use
- Use `runtime.SetFinalizer()` for safety in Go wrappers

### Error Handling

- Return NULL for string functions on error
- Return -1 or 0 for int functions on error
- Go wrapper converts to Go errors with descriptive messages

## Security Considerations

1. **Accessibility Permission:** User must explicitly grant permission
2. **Privacy:** Only capture terminal content, not other apps
3. **Sandboxing:** May need entitlements for App Store distribution
4. **Code Signing:** Required for accessibility access on macOS 10.15+
