# Task 2.8: Linux Terminal Adapter (AT-SPI2)

**Phase:** 2 - Terminal Integration  
**Week:** 9-10 (per implementation plan)  
**Priority:** High  
**Dependencies:** Task 2.1 (Terminal types), Task 2.3 (Detector)

## Objective

Create a `TerminalAdapter` implementation for Linux GNOME Terminal/Konsole using AT-SPI2 (Assistive Technology Service Provider Interface) CGO bindings. This enables automatic terminal content capture on Linux without requiring tmux.

---

## Requirements

### 1. AT-SPI2 Header File (`atspi2.h`)

Create C header with D-Bus/AT-SPI2 function declarations:

```c
// Initialize AT-SPI2 connection
int atspi2_initialize(void);
int atspi2_is_available(void);

// Get terminal window information
char** get_terminal_window_titles(int* count);
int* get_terminal_window_pids(int* count);

// Extract text from terminal window
char* extract_text_from_window(int pid);
char* extract_text_from_focused_terminal(void);

// Memory management
void free_string_array(char** array, int count);
void free_string(char* str);
```

### 2. AT-SPI2 Implementation (`atspi2.c`)

Implement functions using AT-SPI2/D-Bus APIs:

**Key APIs:**
- `atspi_init()` - Initialize AT-SPI2
- `atspi_get_desktop()` - Get desktop accessible object
- `atspi_accessible_get_name()` - Get window title
- `atspi_accessible_get_child_at_index()` - Navigate accessibility tree
- `atspi_text_get_text()` - Extract text content

**Functions to implement:**

1. **`atspi2_initialize()`** - Initialize AT-SPI2, return 0 on success
2. **`atspi2_is_available()`** - Check if AT-SPI2 daemon is running
3. **`get_terminal_window_titles()`** - Find GNOME Terminal/Konsole windows
4. **`get_terminal_window_pids()`** - Get process IDs
5. **`extract_text_from_window(pid)`** - Extract text via AT-SPI2 text interface
6. **`extract_text_from_focused_terminal()`** - Get focused terminal content
7. **Memory management functions**

### 3. Go CGO Wrapper (`adapter_linux.go`)

Create Go wrapper:

```go

// GetTerminalWindowPids returns PIDs of terminal windows
func GetTerminalWindowPids() ([]int, error)
```

### 4. Adapter Implementation (`adapter_linux.go`)

Implement `TerminalAdapter` interface:

```go
// Adapter implements TerminalAdapter for Linux terminals
type Adapter struct {
    config Config
}

// Name returns "linux"
func (a *Adapter) Name() string { return "linux" }

// Available checks AT-SPI2 availability
func (a *Adapter) Available(ctx context.Context) bool {
    return IsATSPiAvailable()
}

// ListSessions returns detected terminal windows
func (a *Adapter) ListSessions(ctx context.Context) ([]DetectedTerminal, error)

// Capture extracts terminal content
func (a *Adapter) Capture(ctx context.Context, terminalID string) (string, error)

// Subscribe streams terminal events
func (a *Adapter) Subscribe(ctx context.Context, terminalID string) (<-chan TerminalEvent, error)

// GetDimensions returns terminal dimensions
func (a *Adapter) GetDimensions(ctx context.Context, terminalID string) (int, int, error)
```

## Implementation Details

### AT-SPI2 Text Extraction

Terminal emulators expose content via AT-SPI2 Text interface:

1. Get desktop: `atspi_get_desktop(0)`
2. Find terminal applications (role = `ROLE_TERMINAL`)
3. Get text interface: `atspi_accessible_get_interface(accessible, "Text")`
4. Extract text: `atspi_text_get_text(text, 0, -1)`

### Supported Terminals

- **GNOME Terminal** - Full AT-SPI2 support
- **Konsole** - AT-SPI2 support
- **Terminator** - AT-SPI2 support
- **xterm** - Limited/no AT-SPI2 support

### Session ID Format

```
linux-{pid}-{window-title}
Example: linux-12345-user@host:~
```

## Testing Strategy

### Unit Tests
- Test adapter Name(), Available()
- Test error handling on non-Linux

### Integration Tests (Linux only)
1. Open GNOME Terminal with content
2. Create adapter: `a := NewAdapter(DefaultConfig())`
3. List sessions: `sessions := a.ListSessions(ctx)`
4. Capture: `content := a.Capture(ctx, sessions[0].ID)`
5. Verify content matches terminal

### Manual Testing
```bash
# Install AT-SPI2 development files
sudo apt install libatspi2.0-dev

# Build
go build ./internal/terminal/...

# Test
go test -v ./internal/terminal/...
```
