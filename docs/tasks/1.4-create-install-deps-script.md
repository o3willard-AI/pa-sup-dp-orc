## Description

Create a bash script `scripts/install-deps.sh` that installs all project dependencies: Wails CLI, Go dependencies, and npm/frontend dependencies.

This task matters because it automates the development environment setup, ensuring all team members and CI systems have consistent dependencies.

## Inputs

**Existing Files:**
- `go.mod` - Go module with Wails v2 dependency
- `frontend/package.json` - Node.js dependencies from Wails init
- `wails.json` - Wails configuration

**Reference:** IMPLEMENTATION_PLAN.md Task 1.4


## Implementation Steps

1. **Create scripts directory**
   ```bash
   mkdir -p /home/sblanken/code/paa5/scripts
   ```

2. **Create install-deps.sh** with:
   - Check for required tools (go, npm)
   - Install Wails CLI: `go install github.com/wailsapp/wails/v2/cmd/wails@latest`
   - Download Go dependencies: `go mod download`
   - Install frontend dependencies: `cd frontend && npm install`
   - Verify: `wails version`, `npm list --depth=0`

3. **Make script executable**
   ```bash
   chmod +x scripts/install-deps.sh
   ```

4. **Test the script**
   ```bash
   ./scripts/install-deps.sh
   ```

## Constraints & Gotchas

- **Use `go install` not `go get`** (deprecated for CLI tools)
- **Pin Wails version** with `@latest` or specific version
- **Check for npm/go** before running (helpful error if missing)
- **Cross-platform** - script is bash (Windows users need WSL or Git Bash)
- **Idempotent** - safe to run multiple times
