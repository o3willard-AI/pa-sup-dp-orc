## Features Implemented

### TerminalAdapter Interface
- ✅ `Name()` - Returns "tmux"
- ✅ `Available(ctx)` - Checks if tmux binary is in PATH
- ✅ `Connect(ctx)` - Validates tmux availability
- ✅ `Disconnect(ctx)` - Closes connection
- ✅ `IsConnected()` - Returns connection status
- ✅ `ListSessions(ctx)` - Lists active tmux sessions
- ✅ `ListPanes(ctx, sessionID)` - Lists panes in a session
- ✅ `Capture(ctx, paneID)` - Captures pane content
- ✅ `Subscribe(ctx, paneID)` - Streams pane updates (polling-based)
- ✅ `GetDimensions(ctx, paneID)` - Returns rows/cols
- ✅ `SendCommand(ctx, paneID, command)` - Sends commands to pane

### Configuration
- Configurable poll interval (default: 500ms)
- Configurable tmux binary path (default: "tmux")
- Socket name support for custom tmux sockets

## Usage Example

```go
package main

import (
    "context"
    "github.com/pairadmin/pairadmin/internal/terminal/tmux"
)

func main() {
    config := tmux.DefaultConfig()
    adapter := tmux.NewAdapter(config)
    ctx := context.Background()
    
    // Connect to tmux
    if err := adapter.Connect(ctx); err != nil {
        log.Fatal(err)
    }
    defer adapter.Disconnect(ctx)
    
    // List sessions
    sessions, _ := adapter.ListSessions(ctx)
    for _, s := range sessions {
        fmt.Printf("Session: %s\n", s.Name)
    }
    
    // Capture pane content
    content, _ := adapter.Capture(ctx, "session-name")
    fmt.Println(content)
    
    // Send command
    adapter.SendCommand(ctx, "session-name", "ls -la")
}
```
