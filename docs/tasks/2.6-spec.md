# Task 2.6: macOS Terminal Adapter Implementation

**Phase:** 2 - Terminal Integration  
**Week:** 9-10 (per implementation plan)  
**Priority:** High

## Objective

Create a `TerminalAdapter` implementation for macOS Terminal.app that uses the CGO bindings from Task 2.5. This adapter will integrate with the Detector service to automatically capture terminal content from Terminal.app windows.

---

## Requirements

### 1. Adapter Implementation (`adapter.go`)

Create `internal/terminal/macos/adapter.go` that implements the `TerminalAdapter` interface from Task 2.1:

```go
type TerminalAdapter interface {
    Name() string
    Available(ctx context.Context) bool
    ListSessions(ctx context.Context) ([]DetectedTerminal, error)
    Capture(ctx context.Context, terminalID string) (string, error)
    Subscribe(ctx context.Context, terminalID string) (<-chan TerminalEvent, error)
    GetDimensions(ctx context.Context, terminalID string) (int, int, error)
}
```

### 2. Adapter Structure

```go
type Adapter struct {
    config       Config
    running      bool
    mu           sync.RWMutex
    pollInterval time.Duration
}

type Config struct {
    PollIntervalMs int
}
```

### 3. Required Methods

**Name()** - Return "macos"

**Available(ctx)** - Check if running on macOS and accessibility is enabled

**ListSessions(ctx)** - Get all Terminal.app windows using CGO bindings:
- Call `GetTerminalWindowTitles()` to get window names
- Call `GetTerminalWindowPids()` to get PIDs
- Return as `[]DetectedTerminal`

**Capture(ctx, terminalID)** - Capture terminal content:
- Use terminalID to find the window
- Call `ExtractTextFromFrontmostTerminal()` or `ExtractTextFromWindow(pid)`
- Return content string

**Subscribe(ctx, terminalID)** - Stream terminal events:
- Start goroutine that polls at configured interval
- Compare content to detect changes
- Emit `TerminalEvent` on changes
- Return event channel

**GetDimensions(ctx, terminalID)** - Get terminal dimensions:
- Return approximate rows/cols (macOS doesn't expose exact dimensions via Accessibility API)
- Can use heuristic based on content lines

### 4. Integration with Detector

The adapter must:
- Work with `Detector` from `internal/terminal/detector.go`
- Be passable to `NewDetector(config, adapters...)`
- Emit events that Detector can process

### 5. Error Handling

Define adapter-specific errors in `errors.go` (or extend existing):
- `ErrNotOnMacOS` - Running on non-macOS platform
- `ErrAccessibilityDisabled` - Permission not granted
- `ErrWindowNotFound` - Specified window doesn't exist
- `ErrCaptureFailed` - Text extraction failed

## Implementation Details

### Session ID Format

Use format: `macos-{pid}-{window-title}`

Example: `macos-12345-main`

### Polling Strategy

```go
func (a *Adapter) Subscribe(ctx context.Context, terminalID string) (<-chan TerminalEvent, error) {
    events := make(chan TerminalEvent)
    go func() {
        defer close(events)
        ticker := time.NewTicker(time.Duration(a.config.PollIntervalMs) * time.Millisecond)
        defer ticker.Stop()
        
        var lastContent string
        for {
            select {
            case <-ctx.Done():
                return
            case <-ticker.C:
                content, err := a.Capture(ctx, terminalID)
                if err != nil {
                    continue
                }
                if content != lastContent {
                    events <- TerminalEvent{
                        Type: EventContentUpdate,
                        TerminalID: terminalID,
                        Data: content,
                    }
                    lastContent = content
                }
            }
        }
    }()
    return events, nil
}
```

### Available() Check

```go
func (a *Adapter) Available(ctx context.Context) bool {
    if runtime.GOOS != "darwin" {
        return false
    }
    return macos.IsAccessibilityEnabled()
}
```

## Testing Strategy

### Unit Tests (adapter_test.go)

```go
func TestAdapterName(t *testing.T) {
    a := NewAdapter(DefaultConfig())
    if a.Name() != "macos" {
        t.Errorf("Expected 'macos', got %s", a.Name())
    }
}

func TestAdapterAvailable(t *testing.T) {
    a := NewAdapter(DefaultConfig())
    available := a.Available(context.Background())
    // On macOS with permission: true
    // On Linux or without permission: false
    t.Logf("Available: %v", available)
}

func TestAdapterListSessions(t *testing.T) {
    a := NewAdapter(DefaultConfig())
    sessions, err := a.ListSessions(context.Background())
    if err != nil {
        t.Logf("ListSessions error (expected if no Terminal.app): %v", err)
    }
    t.Logf("Found %d sessions", len(sessions))
}
```

### Integration Test

1. Open Terminal.app with some content
2. Create adapter: `a := NewAdapter(DefaultConfig())`
3. List sessions: `sessions := a.ListSessions(ctx)`
4. Capture content: `content := a.Capture(ctx, sessions[0].ID)`
5. Verify content matches terminal
