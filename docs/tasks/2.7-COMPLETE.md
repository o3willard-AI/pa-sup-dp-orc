### Key Features

1. **Multi-Adapter Support:**
   - `NewDetector(config, adapters...)` - Create with adapters
   - `AddAdapter(adapter)` - Add at runtime
   - `GetAdapters()` - Get registered adapters

2. **Auto-Registration (adapters package):**
   - `NewDetectorWithAutoAdapters(config)` - Platform-specific registration
   - tmux: Always registered
   - macOS: Registered when accessibility enabled

3. **Adapter Status:**
   - `GetAdapterStatus()` - Returns `[]AdapterStatus`
   - Fields: Name, Available, Type (multiplexer/native)

4. **Accessibility (macOS):**
   - `RequestAccessibilityPermission()` - Request permission
   - `IsAccessibilityEnabled()` - Check status

### Import Cycle Resolution

Created separate `internal/adapters` package:
```
terminal (interfaces, types, detector)
    ↑
tmux, macos (adapter implementations)  
    ↑
adapters (factory, registration)
```

## Usage Example

```go
import (
    "github.com/pairadmin/pairadmin/internal/adapters"
    "github.com/pairadmin/pairadmin/internal/terminal"
)

// Create detector with auto-registered adapters
det := adapters.NewDetectorWithAutoAdapters(
    terminal.DefaultDetectorConfig()
)

// Check adapter status
statuses := det.GetAdapterStatus()
for _, s := range statuses {
    fmt.Printf("Adapter: %s, Available: %v\n", s.Name, s.Available)
}

// Request permission if needed (macOS only)
if !adapters.IsAccessibilityEnabled() {
    adapters.RequestAccessibilityPermission()
}

// Start detection
det.Start()
defer det.Stop()

// Subscribe to events
for event := range det.Events() {
    fmt.Printf("Event: %s - %s\n", event.Type, event.TerminalID)
}
```
