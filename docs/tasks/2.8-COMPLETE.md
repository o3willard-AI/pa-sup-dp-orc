1. **AT-SPI2 C Interface:**
   - `atspi2_initialize()` - Initialize AT-SPI2
   - `atspi2_is_available()` - Check availability
   - `get_terminal_window_titles()` - Find terminal windows
   - `extract_text_from_window()` - Extract text content
   - `get_terminal_dimensions()` - Get rows/cols

2. **Go Adapter Implementation:**
   - `Name()` - Returns "linux"
   - `Available()` - Checks AT-SPI2 availability
   - `ListSessions()` - Lists terminal windows
   - `Capture()` - Extracts terminal content
   - `Subscribe()` - Streams content updates
   - `GetDimensions()` - Returns terminal size

3. **Session ID Format:**
   ```
   linux-{pid}-{window-title}
   Example: linux-12345-user@host:~
   ```

## Usage Example

```go
import (
    "github.com/pairadmin/pairadmin/internal/terminal"
)

// Create Linux adapter
adapter := terminal.NewAdapter(terminal.DefaultConfig())

// Check availability
if adapter.Available(ctx) {
    // List terminal sessions
    sessions, _ := adapter.ListSessions(ctx)
    
    // Capture content from first session
    if len(sessions) > 0 {
        content, _ := adapter.Capture(ctx, sessions[0].ID)
        fmt.Println(content)
    }
}
```
