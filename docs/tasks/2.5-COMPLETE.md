### About macOS Accessibility
1. **Permission Model:** Requires explicit user grant via System Settings
2. **AXUIElement API:** Core Foundation based, requires careful memory management
3. **Text Extraction:** Requires navigating window â†’ textarea â†’ value attribute tree

### About CGO
1. **Objective-C Bridging:** Use extern "C" for C-compatible headers
2. **Framework Linking:** #cgo LDFLAGS for Cocoa, ApplicationServices
3. **Build Tags:** Essential for platform-specific code
4. **Memory Safety:** unsafe.Slice for C array to Go slice conversion
