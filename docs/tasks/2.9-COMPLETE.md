1. **Subpackage Pattern:** Follows the same pattern as `macos/` subpackage for platform-specific code
2. **Build Tags:** Proper `//go:build windows` and `//go:build !windows` tags for cross-compilation
3. **CGO Stub:** C files provide stub implementations; full implementation requires COM/UI Automation integration
4. **Interface Compliance:** Implements `TerminalAdapter` interface with all required methods:
   - `Name()` - Returns "windows"
   - `Available(ctx)` - Checks UI Automation availability
   - `ListSessions(ctx)` - Returns detected terminal windows
   - `Capture(ctx, terminalID)` - Extracts terminal content
   - `Subscribe(ctx, terminalID)` - Streams terminal events
   - `GetDimensions(ctx, terminalID)` - Returns rows/cols
   - `Stop()` - Stops the adapter
