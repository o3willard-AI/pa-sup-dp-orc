## Description

Update the `.gitignore` file created by Wails init to include comprehensive patterns for Go binaries, build artifacts, node_modules, OS files, and editor configs.

This task matters because a proper `.gitignore` prevents committing generated files, binaries, and environment-specific configs to version control.
